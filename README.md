## Features

### Add New Event

* Enter event title
* Select date
* Choose category (Conference, Workshop, Meetup, Webinar, Social)
* Add description
* Click **Add Event** to create a new event card

### My Events Section

* Displays all added events as cards
* Each card shows:

  * Event title
  * Date
  * Category
  * Description
* Delete button to remove event instantly
* "Clear All Events" button
* "Add Sample Event" button 

* ⌨️ **Keyboard detection feature**
  Shows which key user presses in real time using JavaScript.
